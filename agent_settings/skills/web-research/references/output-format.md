# Output Format

Use this structure unless the user asks for a different format.

## Default Response Shape

- Direct answer in 1 ~ 3 sentences
- Key facts
- Source links
- Uncertainty or caveats

## Required Elements

- State the date when the answer depends on current information.
- Distinguish confirmed facts from inference.
- Include links for every material claim.
- Keep quotes short and use paraphrase for the rest.

## Suggested Patterns

### Fast fact check

- Answer
- Verified on: `<date>`
- Sources: `<link 1>`, `<link 2>`

### Comparison

- Best short summary
- Side-by-side differences
- Source links by item
- What is still unclear

### Technical research

- What the docs say
- Version or release date
- Migration or compatibility note
- Links to primary docs

### News summary

- What happened
- When it happened
- Why it matters
- Links to reporting and any primary statement
